skills

HTML

Hypertext Markup Language is the foundational language used to create the structure and content of a webpage. It forms the backbone of every website by defining elements like text, images, links, headings, and more. In essence, HTML provides the *skeleton* of a webpage.

CSS

Cascading Style Sheetsis the language used to style and layout web pages. While HTML provides the structure and content, *CSS controls the presentation*, determining how the elements look and feel to users. It allows you to apply colors, fonts, spacing, positioning.

JavaScript

JavaScript is a versatile, high-level programming language that enables dynamic and interactive functionality on websites. It plays a crucial role in web development, complementing HTML and CSS to create a complete web experience. While HTML structures the content and CSS styles.
